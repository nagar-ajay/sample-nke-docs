What is Kubeflow on NKE?

The Kubeflow on NKE project is an open source distribution of Kubeflow, dedicated to making deployments of machine learning (ML) workflows on Nutanix Kubernetes Engine simple, portable and scalable. Kubeflow on NKE project provides seamless integration with Nutanix services.
